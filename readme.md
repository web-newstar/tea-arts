# 如何使用？？？
  ### 打开命令行 快捷方式（window+r）依次执行以下步骤
    * npm install webpack -g --save
    * npm install
    * npm run start
    * npm run watch
    * 打开浏览器 访问localhost:4000/index
  ### 前端路由
    * 主页 /index
    * 关于茗悦 /about
    * 最新动态 /new
    * 培训课程 /class
    * 讲座表演 /perform
    * 茶学堂 /school
    * 联系我们 /contact
  ### 后台路由
    * 主页 /admin/index
    * 关于茗悦 /admin/about
    * 最新动态 /admin/new
    * 培训课程 /admin/class
    * 讲座表演 /admin/perform
    * 茶学堂 /admin/school
    * 联系我们 /admin/contact
    * 在线报名/admin/apply
